День 1. Мистер Безье.

transition-timing-function свойства:

  • ease - медленно-быстро-медленно ( по умолчанию)
  • linear - действует одинаково на протяжении всей анимации.
  • ease-in - медленно-быстро
  • ease-out - быстро-медленно
  • ease-in-out - медленно-медленно
  • step-start - без анимации сразу применяет конечные стили
  • step-end - без анимации применяет стили в конце.
  • steps - указывает конкретное количество шагов для анимации
  • cubic-bezier - задает анимацию по кривой Безье
transition: width 3s ease
transition: width 3s linear
transition: width 3s ease-in
transition: width 3s ease-out
transition: width 3s ease-in-out
transition: width 3s step-start
transition: width 3s step-end
transition: width 3s steps(5)
cubic-bezier(.05, .98, .94, .06)
cubic-bezier(0,1.91,1,-0.96)
cubic-bezier(0,-0.45,.98,-0.33)
cubic-bezier(0,1.50,0,1.50)

На этом сайте вы сможете сгенерировать собственную кривую Безье. http://cubic-bezier.com/

День 3. Занимацкий CSS

Css3 поддерживает покадровую анимацию, которая позволяет создавать сложные анимации обьектов.

Достигается она при помощи свойства - animation и правила @keyframes.

При помощи свойства мы вызываем анимацию, а в правиле мы его задаем и описываем, как оно должно работать.

Пример animation

У animation 2 обязательных параметра - имя и ее продолжительность.

Мы можем задавать анимацию в @keyframes двумя способами.

Первый - это простая анимация( from / to )

Второй - покадровая анимация, которую мы задаем при помощи ( процентов 0 - 100 )

Пример покадровой анимации:

0% animation 100%

Парочка советов:

1. Чтобы в конце не было прыжка, делайте последний слайд такой же, как и оригинальное состояние блока.

2. А чтобы вернуть оригинальное состояние - можно просто указать пустые фигурные скобки.

3. Можно удалить одинаковые свойства, которые находятся в промежуточных кадрах.

4. Анимация не должна двигать другие элементы сайта. Поэтому готовьте для нее плацдарм.

5. Если вы хотите двигать обьекты, используйте позиционирование.

5. Если вы используете позиционирование, все свойства, которые вы будете указывать, должны уже быть заданы вашему блоку.

День 4. Анатомия анимации.

Свойство animation имеет много различных настроек для анимации обьектов.

Трепанация анимации:

animation-name - задает имя для анимации.

animation-duration - задает время анимации.

animation-timing-function - кривая Безье, котора задает стиль анимации.

animation-delay - задержка.

animation-iteration-count - счётчик(используем целые числа), infinite - бесконечно.

animation-direction - указывает, как должна выполняться анимация.

  • normal - анимация работает от начала и до конца(от 0 - к 100%).
  • reverse - анимация работает от конца и до начала (от 100% - к 0).
  • alternate - анимация идет от начала к концу и, снова к возвращается в начало (от 0 - до 100%, и от 100% - к 0).
  • alternate-reverse - анимация работает наоборот
    ( со 100% - к 0 и от 0 - к 100%)

Альтернативная анимация будет работать только если указан повтор больше одного.

animation-fill-mode - определяет как будет анимация воздействовать на элемент, до и после анимации.

  • none - после анимации элемент возвращается на круги своя.
  • forwards - после анимации элемент застывает в последнем кадре.
  • backwards - анимация начинает работать до начала анимации.
  • both - начинает работать сразу и застывает в последнем положении.
test 2s ease-in 2s 2 alternate both;

animation-play-state - отвечает за паузу.

running - анимация работает; paused - пауза.

При использовании animation Важно соблюдать порядок параметров!

При создании анимации обязательно должны быть указаны два параметра: animation-name и animation-duration !

День 5. Подготовка к Flex'у.

RESET CSS - сброс стилей.

Полностью обнуляет все стили по умолчанию у браузеров. Убирает отступы, поля, маркеры, межстрочные элементы и остальное оформление по умолчанию у браузеров.

Придется полностью воссзоздавать все стили с нуля.

NORMALIZE CSS - нормализация стилей.

В отличии от сброса стилей приводит все элементы сайта к единому нормальному виду во всех браузерах.
