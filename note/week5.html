День 1. Атрибуты тега input

При помощи дополнительных атрибутов, мы можем более конкретно настраивать поля наших форм.

Атрибуты тега input

  • required - делает поле обязательным для заполнения.
  • placeholder - текст в поле, который исчезает при заполнении формы.
  • autocomplete - автозаполнение поля.
  • pattern - задает формат ввода данных для поля.
  • min -задает минимальное значение для поля.
  • max - задает максимальное значение для поля.
  • multiple - разрешает указать несколько почтовых адресов (через запятую), или файлов для загрузки.
  • step - задает шаг и актуален для числовых полей и ползунка.
  • autofocus - задает автоматический фокус для указанного поля.
  • form - привязывает поле к определенной форме.
  • list - используется для привязки к тегу <datalist>, что бы указать на список вариантов.

День 2. Человекоформа :)

<label> - этот тег связывает текст и поле формы. То есть, при клике на тексте, активируется поле.

Связываться можно двумя способами. Первый - разместить все внутри тега <label>. Второй при помощи специального атрибута и идентификатора.

<select> - задает список в котором можно выбрать определенные значения. Значения (пункты списка) задаются при помощи тега <option>

<optgroup> - обьединяет пункты списков в отдельные группы.

<output> - через этот тег можно вывести результат поля формы или его дополнительно обработать при выводе.

<datalist> - создает список вариантов для заполнения поля. Активируется данный список при фокусе на поле (двойной щелчок) либо при вводе информации.

<textarea> - поле для ввода большого текта. Текст виден в несколько строк.

Атрибуты поля <textarea>:

  • maxlength - задает максимальное количество символов в поле.
  • wrap
    • soft - Длинный текст, который самостоятельно не помещается в поле по ширине, будет автоматически перенесен на новую строку, однако передаваться на сервер будет как одна строка.
    • hard - Слова в поле, как и в первом случае переносятся автоматически, разница только в том, что все переносы сохранятся.
    • off - Нестандартное значение. Переносы строк отключены. При введении длинного текста без переносов, он будет печататься в одну строку, при этом будет отображаться полоса прокрутки.

При значении атрибута wrap равном hard - обязательно должен быть указан атрибут cols.

День 5. Семантические элементы в HTML5.

Тег <main>- в нём должен содержаться главный контент страницы.

  1. Этот тег должен быть уникальным и встречаться один раз на странице.
  2. Он не должен быть внутри других стуктурных тегов.

Если современные браузеры не будут понимать как отображать данный тег, пропишите ему следующие стили:

main {display: block;}

Тег <mark> - создаёт на странице выделенный текст. В некоторых браузерах он выделяется по умолчанию желтым.

Это выделенный текст. Используйте его для важных заметок.

<time> - в нём указываем дату или время, или то и другое.

Атрибуты:

  • datetime - дата и время в тексте.
  • pubdate - указывает дату публикации.

1984.07.20 - дата моего рождения.

<figure> - в нём мы можем группировать какие-либо элементы сайта.

<figcaption> - в нем размещается описание группированных элементов сайта.

Этот тег должен быть внутри тега <figure> - и он должен быть либо самым первым, либо самым последним.
