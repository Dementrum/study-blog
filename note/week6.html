День 1. Прогрессивные личности

Тег <progress> - создает прогресс-бар. Который мы можем анимировать через JavaScript

Загрузка мозга: Загрузка мозга:

<meter> - отображает числовые значения в баре и в зависимости от значения, окрашивает строку в определенный цвет.

Атрибуты:

  • value - текущее значение (состояние) тега meter. Может также указываться в процентах.
  • min - минимальное значение тега meter.
  • max - максимальное значение тега meter.
  • low - определяет значение, которое ниже указанного в этом атрибуте. Это считается низким значением. Например, если задать значение low равным 33, то этот атрибут определяет то значение, которое ниже 33. optimum - определяет наиболее оптимальное значение тега meter.
  • high - определяет наиболее высокое значение тега meter.

Нервное состояние:

хорошее - хорошее

волнуется - волнуется

может ударить - может ударить

Если браузер не понимает теги <progress> и <meter> вместо рисунков он отображает текст, который находится в них.

<details> - создаёт спойлер.

Атрибуты:

  • Указывает, что содержимое тега изначально должно быть показано в развернутом виде. Значения для логического атрибута open можно задавать следующими способами:
<details open>
<details open="open">

<summary> - создаёт название для спойлера.

День 2. Псевдоэлементы.

Это такая фиговина, которая позволяет использовать какие-то элементы, не добавляя тегов в HTML код.

Псевдоэлемент ::after - добавляет псевдоэлемент в конце тега.

Псевдоэлемент ::before - добавляет псевдоэлемент в начале тега.

Когда не нужен текст для псевдоэлементов, вы можете в контенте указать пустой символ или спецсимвол пробела.

Псевдоэлементы ::after и ::before работают при наличии свойства content

Псевдоэлемент ::first-letter - задает стили для первого символа.

Псевдоэлемент ::first-line - задает стили для первого строчки.

::selection - позволяет задавать стили к выделенному тексту.

Лучше использовать новые псевдоэлементы. Так как они работают во всех браузерах.

В браузере мозилла это свойство по умолчанию не работает, однако вы можете использовать ::-moz-selection

День 3. Псевдоклассы.

Псевдоэлемент :first-child - задает стили для первого дочернего элемента своего родителя.

Псевдоэлемент :last-child - задает стили для последнего дочернего элемента своего родителя.

Псевдоэлемент :nth-child(значение) - задает стили через нумерацию дочерних элементов родителя.

Значения псевдокласса:

  • odd - применяет стили к нечетным элементам.
  • even - применяет стили к четным элементам.
  • число - задает значение определенному элементу.
  • 1 - равносилен псевдоклассу :first-child
  • 2n - равносилен значению even.
  • 2n+1 - равносилен значению odd.

Как работает: :nth-child(2n+3)
"2n"- это шаг. Шаг идет через каждые два элемента.
А "+3" - указывает с какого по счету элемента, надо начинать шаг.

Псевдоэлемент :nth-last-child(значение) - задает стили через нумерацию дочерних элементов родителя начиная с конца.

Когда мы ему задаем значение равное 1- оно равносильно псевдоклассу :last-child

Как поставить Browser Sync:

Это такой браузер, который подхватывает ваш код на лету, и вам не нужно постоянно обновлять браузер. Если кто знает, наподобие LiveReload, только гораздо лучше.

Для начала нам надо установить Node.js. Зайдите на сам сайт Node.js. Выберите установщик, версии LTS и установите.Это основа. А дальше я решил показать видео от loftblog, потому как лучше один раз увидеть, чем сто раз услышать

Тут напишу код, чтобы его можно было скопировать и не писать ручками. Вводим в консоли:
npm install -g browser-sync
И еще одна команда:
browser-sync start --server --files "*.*"
Отслеживает все файлы в данной папке.
