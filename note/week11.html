Неделя 11 - Анимация

День 1. Фильтры.

CSS свойство filter устанавливает визуальный эффект или комбинацию эффектов для элемента. К фильтрам относится изменение прозрачности, добавление тени, трансформация и др.

blur - показывает размытость картинки, значение обозначается в пикселях, причем, чем больше пикселей, тем больше размытие.

img.blur {filter: blur(2px);}

brightness - показывает яркость, значения задаются либо в целых или дробных числах, либо в процентах.

img.brightness {filter: brightness(3);}

contrast - показывает контраст, значения задаются либо в целых или дробных числах, либо в процентах.

img.contrast {filter: contrast(1.5);}

saturate - показывает насыщенную теплоту картинки, значения задаются либо в целых или дробных числах,лбо в процентах.

img.saturate {filter: saturate(200%);}

opacity - устанавливат прозрачность картинки, значения также задаются в процентах или в дробных числах, но есть ограничения: дробные числа - от 0 до 1, в процентах - от 1 до 100.

img.opacity {filter: opacity(50%);}

invert - инверсия цвета, означает противоположный - любой цвет станет противоположным по спектру, значения задаются в процентах.

img.invert {filter: invert(100%);}

sepia - сепия дает набор оттенков от темно-коричневого до светло-золотистого тона, цвета сепии ассоциируются со стариной, значения задаются либо в целых или дробных числах, либо в процентах.

img.sepia {filter: sepia(0,5);}

grayscale - градация серых тонов, полутонов, отличительной особенностью которой является отсутствие контраста по цвету. Цветовой режим изображений, которые отображаются в оттенках серого цвета. Значения также задаются в процентах или в дробных, либо в целых числах, но есть ограничения: дробные числа -от 0 до 1, в процентах -от 1 до 100.

img.grayscale {filter: grayscale(0,7);}

hue -rotate - меняет цвета изображения в зависимости от заданного угла. Угол поворота определякт на сколько изменится данный цвет в цветовом круге от красного до фиолетового. Значение задается в градусах(0deg -360 deg)

img.hue-rotate {filter: hue-rotate(270deg);}

drop-shadow - добавляет тень, а также ее направление и цвет, задается четырьмя значениями - по горизонтали, вертикали, радиус и цветовой оттенок тени.

img.drop-shadow {filter: drop-shadow(5px 5px 20px #111);}

Все эти свойства можно комбинировать. Можно посмотреть тут: Фильтры

День 2. Animate CSS.

Сегодняшний день был связан с анимациями картинок с помощью готовых скриптов

Скрипты для анимации мы выбирали вот тут: Скрипты

Алгоритм подключения скриптов:

  1. Подключаем файл стилей

    2. <link rel="stylesheet" href="animate.css">

  2. Добавляем класс animated элементу, который нужно анимировать. Можно подключить класс infinite , для работы по кругу
  3. Выбираем и добавляем эффект анимации в виде класса, например для заголовка

    4. <h1> class="animfted wobble">Animate CSS </h1>

  4. Для анимации при прокрутке страницы необходимо подключить javascript и библиотеку jquery. Код скрипта представлен ниже 
    5.                     <script>
                     $(window).scroll(function()){
                      var imagePos = $(this).offset().top;
                      var topOfWindow = $(window).scrollTop();
                      if (imagePos < topOfWindow+100){
                       $(this).addClass('bounceIn');
                      }
                     });
                    });
                    <script>
  5. Последнюю версию Jquery можно скачать с главной страницы официального сайта Jquery.com 
    6.               <head>
              <script type="text/javascript" src="/js/jquery-3.3.1.js">
              <head>

День 3. WOW.js

Для анимации элементов сайта при прокрутке страницы можно применять специальную JS библиотеку под названием WOW.js. Она независима от jQuery, как большинство библиотек.Очень проста в подключении и настройке, имеет маленький размер и не создает видимой нагрузки на браузер. Благодаря WOW.js, мы можем управлять анимацией через специальные атрибуты, в том числе временем, задержкой и повторением.

Подключение и настройка.

Данная библиотека работает совместно с наборои анимации Animate.css, поэтому нужно подключать файл стилей. Затем подключение скрипта самой библиотеки и скрипта ее инициализации. Код подключения представлен ниже.

              <link rel="stylesheet" href="animate.css">
              <script src="wow.min.js"><script>

Инициализация скрипта...

             <script>
              <new WOW().init();
             <script>

Дальше идёт настройка анимации выбранного элемента. Добавляем класс WOW и класс нужного эффекта анимации к элементу. Например:

            <img class="wow fadeInLeft" src="image/dog1.jpg alt="Собака-лучший друг человека">

Дополнительные настройки WOW.js скрипта:

  • data-ow-duration-означает, сколько будет длиться анимация, время анимации обозначается в секундах.
  • data-wow-delay- предстартовая задержка анимации, также обозначается в секундах.
  • data-wow-offset- обозначает, когда картинка начнет проигрываться при прокрутке ползунка, обозначается в числах, например, <data-wow-offset="200">
  • data-wow-iteration- обозначает количество повторений анимации.

Применение дополнительных настроек:

            <img class="wow fadeinleft" data-wowo-duration="1.5s"data-wow-delay="3s">
            <img class="woa fadeInLeft"data-wow-offset="100"data-wow-iteration="3">

Инфу по скрипту WOW.js можно посмотреть здесь: WOW.js

День 4. Waypoints

Waypoints - это библиотека Js, работающая в паре с JQuery/Zepto, и с третьей версии, по заявлению автора, не имеет жестких зависимостей и работает самостоятельно. При помощи ее можно добиться анимационных эффектов над элементами при прокрутке страницы, и сегодня будет рассмотрен тот же эффект, что и при использовании библиотеки WOW.js, который был рассмотрен выше

Информация о скрипте Waypoints можно посмотреть здесь

Для реализации задачи будут использованы сборка анимационных эффектов animate.css, а также библиотека jQuery, подключенная через Яндекс-сервис.

Чтобы все заработало нужно подключить следующие скрипты JQuery и Waypoints перед закрывающим тегом </body>
